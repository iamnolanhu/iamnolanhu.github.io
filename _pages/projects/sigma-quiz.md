An interactive quiz platform with Python backend and responsive frontend.

## Overview

A full-stack quiz application that allows users to take interactive quizzes with a modern, responsive interface built with React and powered by a Python backend.

## Tech Stack

**Frontend**
- React for dynamic UI
- Responsive design
- Interactive quiz components

**Backend**
- Python API server
- Quiz logic and scoring
- Question management

## Features

- Multiple choice questions
- Score tracking
- Timer functionality
- Results summary
- Responsive design for all devices
